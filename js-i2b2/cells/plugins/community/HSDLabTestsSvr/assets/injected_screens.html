Healthcare System Dynamics: Laboratory Tests

This plugin demonstrates how Healthcare System Dynamics can aid in the interpretation of laboratory test results. For example, counterintuitively, patients with a normal WBC value at 4am have a lower three-year survival rate than patients with an abnormal WBC value at 4pm. The reason is because clinicians only order tests at 4am for patients who they think are sick. Often HSD dimensions of a laboratory test, such as the time of day or the day of the weeek the test was ordered is more predictive of patient outcomes than the value of the test result.

To use this plugin, first select a patient set to serve as the base cohort. Then, select a laboratory test and an "outcome" concept (e.g., a diagnosis, procedure, etc.). Finally, click the View Results tab to visualize the fraction of patients with the outcome concept, broken down by both the value of the test result and the HSD dimension of the test result.
