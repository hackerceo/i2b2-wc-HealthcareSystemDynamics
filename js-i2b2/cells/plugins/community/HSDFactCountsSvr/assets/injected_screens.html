Healthcare System Dynamics: Fact Count

This plugin graphs the "fact count" distribution of one or more patient sets. Fact count is the total number of diagnoses, procedures, medications, test results, etc., in a patient's medical record. It is a measure of the amount of interaction the patient has had with the healthcare system. When comparing patient cohorts, it is important to control for fact count, in the same way that one might control for patient age, sex, or race.

Drop one Patient Set on the input box below to act as the baseline population. Fact count percentiles for this patient set will be depicted as a series of parallel curves.
